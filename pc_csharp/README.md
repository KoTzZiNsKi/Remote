# Remote

Удалённое управление компьютером с телефона или планшета: мышь, клавиатура, буфер обмена, медиаклавиши и действия питания (сон, выход, перезагрузка, выключение).

## Как это работает

На компьютере запускается приложение Remote (ПК). Оно поднимает TCP-сервер для приёма команд и опционально UDP для обнаружения в локальной сети. Телефон и ПК должны быть в одной сети (один Wi‑Fi или общий маршрутизатор). В приложении на телефоне вводите IP-адрес ПК и порт (по умолчанию 1978), при необходимости — пароль. Подключение по TLS не используется, трафик идёт по локальной сети.

После подключения с телефона можно двигать курсор жестом, нажимать кнопки мыши, листать, вводить текст (в том числе через виртуальную клавиатуру), копировать и вставлять буфер обмена, регулировать громкость и отправлять команды питания.

## Приложение на ПК (Windows)

- Работа в трее: после запуска окно можно закрыть, приложение остаётся в системном трее.
- Настройки: имя ПК, модель, порт TCP, пароль подключения, запуск вместе с Windows, тихий режим.
- Пароль: можно установить, изменить или сбросить. Без пароля подключиться может любой в сети; с паролем — только тот, кто его знает.
- Тихий режим: при включении приложение не отвечает на широковещательные запросы поиска по сети. ПК не будет виден в списке устройств при автоматическом поиске, но к нему по-прежнему можно подключиться вручную по IP и порту (или по QR-коду из настроек).
- IP и QR-код: в настройках или через пункт в трее можно открыть окно с текущим IP, портом и QR-кодом для быстрого ввода данных на телефоне. Если у ПК несколько сетевых адресов, отображаются все, переключение стрелками.
- Обновления: пункт «Проверить обновления» в меню трея сверяет версию с релизами на GitHub. Если вышла новая версия, она скачивается, текущий exe заменяется, приложение перезапускается. Если версия уже последняя, показывается сообщение об этом.
- Справка: в меню трея — «Начало работы» (краткий пошаговый помощник) и ссылка на этот репозиторий как FAQ.

## Приложение на телефоне (Android)

- Подключение по IP и порту (и при необходимости по QR-коду с ПК).
- Экран мыши: жест для движения курсора, тап — левая кнопка, долгий тап — правая кнопка, прокрутка.
- Клавиатура: ввод текста и отправка на ПК.
- Дополнительно: громкость вверх/вниз/без звука, буфер обмена, команды питания (сон, выход, перезагрузка, выключение).

## FAQ

**Компьютер не находится в списке на телефоне.**  
Убедитесь, что телефон и ПК в одной сети. Если на ПК включён тихий режим, автоматический поиск не сработает — введите IP и порт вручную или отсканируйте QR-код из окна «IP и QR-код» на ПК.

**При подключении пишет «Неверный пароль».**  
На ПК в настройках задан пароль — введите тот же пароль в приложении на телефоне. Пароль можно сбросить в настройках на ПК.

**Подключение обрывается.**  
Проверьте, что на ПК не уходит в сон сетевая карта и что брандмауэр Windows не блокирует входящие подключения на выбранный порт (по умолчанию 1978).

**Как обновиться до новой версии?**  
Правый клик по иконке в трее — «Проверить обновления». Если есть новая версия, она скачается и установится сама. Если нет — появится сообщение, что установлена последняя версия.

Сборки Android публикуются в разделе Releases этого репозитория или в описании проекта.
